# भाग १ — आरम्भिका

## 1.1 लिपिः एवं एन्कोडिङ्गः
- देवनागरी (UTF‑8, NFC)।
- दण्ड `।` वाक्य-समाप्तिः।

## 1.2 अङ्क-प्रणाली
- ०–९; दशमाधारः।
- उदाहरणम्: `स्थापय अ, १२३।`

## 1.3 खण्ड-रचना
- `आरम्भ … समाप्ति` अथवा `{ … }`।

## 1.4 प्रथम कार्यक्रमः
```
कार्य मुख्य(): शून्य आरम्भ
  लिख("नमस्ते")।
समाप्ति
```
